TxtAdvice (OutFitr)

Instant fashion advice

Ever second-guess yourself after putting on an outfit before heading out the door? I know I do! As the lead platform developer of TxtAdvice I worked with a few colleagues of mine to assemble an app aiming to deliver real-time fashion advice to users by connecting them with professional stylists. Send in a question or a picture and you'll have an expert opinion in no time.

Basic Chat

Portable Instant Messaging

Basic Chat functions as a lightweight chat room providing a seamless instant-messaging communication similar to Facebook's. The site also tracks analytics for each user which is available on their profile page. Check out the project on Github! More features to be announced.

Scopium

Multiple-Sclerosis Monitoring Application

Scopium is a project some friends and I created for Hack UMASS III. It makes use of Arduino microcontrollers which transmit serialized data to an iOS application, helping monitor symptoms of Multiple Sclerosis for patients in need. Studies show that MS patients aware of the severity of their symptoms live more comfortably than those who are left in the dark. See more: Devpost.

Wave Calendar

Party Scheduling Social Network

If you've ever been bored and wondered what events are going on around your area, you aren't alone. Wave Calendar provides an easy way for users to find out where the parties (waves) are around them! Once signed up you can create your own wave, browse parties in your local area and rate those you've attended. This web application is built with MEAN stack. Check out the project on Github!

Medicus

Skin Condition Diagnostic Application

Medicus is yet another mobile application my team and I developed at Hack UMASS! It harnesses image recognition to confidently diagnose various skin ailments given a picture of the condition. An efficient diagnostic tool modeled as an iPhone application has the potential to save doctors time as well as mitigate user anxiety by providing them with an immediate, accurate diagnosis. See more: Devpost.
