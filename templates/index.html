Basic Chat

Portable chat messaging

Basic Chat is a small project which implements both Node.js and MongoDB. It allows the user to send instant messages to contacts that are also logged in. The site also tracks analytics for each user which can be viewed on their profile. Check out the project on Github! More features to be announced.

Scopium

Multiple-Sclerosis Monitoring Application

Scopium is a project that a couple of friends and I created for Hack Umass. It makes use of little mechanical Arduino sensors which transmit data to an iOS Appplication, helping monitor signs of multiple sclerosis for patients in need. Studies show that patients more aware of their symptoms are more comfortable than patients unaware of sympotoms they may be experiencing. See more: Devpost.
